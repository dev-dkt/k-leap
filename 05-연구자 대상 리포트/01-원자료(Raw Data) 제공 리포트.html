원자료(Raw Data) 제공 리포트
event 2024년 9월 15일

1. 목적

이 리포트는 한국어 능력 시험에 대한 필기 및 말하기 시험 성적을 기반으로 한 익명화된 원자료를 연구자들에게 제공하는 것을 목적으로 합니다. 연구자들은 이 데이터를 활용하여 통계 분석, 언어 능력 연구, 시험 품질 분석 등의 다양한 연구를 진행할 수 있습니다.

2. 제공 데이터 유형

필기시험 데이터

  • 문항별 응답 데이터
  • 문항별 점수 데이터
  • 영역별 성적 데이터
  • 시험지 정보

말하기 시험 데이터

  • 루브릭 평가 데이터
  • 응답 녹취 및 전사 데이터 (요청 시)

3. 데이터 포맷

CSV 파일 구조 예시

4. 데이터 사용 가이드라인

  • 제공되는 원자료는 익명화 처리되어 개인 정보가 보호됩니다.
  • 데이터는 연구 목적으로만 사용 가능하며, 상업적 사용은 금지됩니다.
  • 연구자는 제공된 데이터를 안전하게 저장하고 관리해야 합니다.

5. 개인정보 보호

  • 모든 데이터는 수험자의 개인정보가 익명화 처리되어 제공됩니다.
  • 연구자는 데이터 사용 동의서를 제출하고 보안 조치에 동의해야 합니다.
  • 데이터는 제공된 연구 목적 이외의 용도로 사용할 수 없습니다.
